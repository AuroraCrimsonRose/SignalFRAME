# SignalFRAME

**Build It. Stream It. Own It.**  
An experimental modular internet radio station framework built by **CATALYSTS LABS**.

SignalFrame was an attempt to build a lightweight, self-hosted platform for managing multiple internet radio stations with per-station configuration, theming, and streaming integration.

The project is incomplete and no longer under active development. Core systems were prototyped in PHP, but the framework is not in a functional or production-ready state.

---

## Concept

SignalFrame explored a modular architecture for internet radio platforms, including:

- Multi-station folder-based structure
- Per-station themes and asset overrides
- REST-style API design for station control
- Token-based access control concepts
- Integration ideas for Icecast / Liquidsoap / AzuraCast
- Separation between hobbyist and commercial feature sets

---

## Status

This project is abandoned and incomplete.

The codebase is not guaranteed to run, and significant portions of intended functionality were never finished or fully implemented.

It is preserved for reference, experimentation, and architectural ideas.

---

## Notes

- Built as an early-stage PHP framework experiment
- Never reached stable or production use
- Some architectural ideas may be reused in future projects
- Not intended for deployment
